You interact with the Service via Github issue comments, called commands.

A command will always be addressed to @charlesmike

More types of commands are understood. Bellow you can find a detailed table with the keywords for each command:


TypeKeywordsExampleEffect
Hello hello @charlesmike hello there!! The bot will answer explaining what it can do for you.
Index site index, site @charlesmike index this site The bot will crawl and index the whole website
Index page index, page @charlesmike pls index [this](link/to/page) page Notice the Markdown styling - it's mandatory. The bot will crawl and index the specified page
Delete index delete @charlesmike delete `repo` index The index will be deleted. Note that you have to specify the repo's exact name (case sensitive) between back-apostrophes. After running this command, you should remove the script from your website since the search won't work anymore

More commands to come. Note that, in order for a command to be recognized properly, all the keywords must be present.


Important

In order to give any command, besides hello, you need to be the repository's owner. This means that:

  • The repo is under your name.
  • The repo is not a fork.
  • If the repo is under an organization, you must be one of the organization's active admins.
    • In this case, you need to invite @charlesmike to be a member of the organization because otherwise it won't be able to check your membership. See here.

About @charlesmike

Charles Michael is a Github chatbot which crawls and indexes your website's content making it searchable. You simply have to add a JS script to your page to render the search box.
